JS kabučių tipai

  1. ' - (viengubos) Jeigu string'e reikia panaudoti dvigubas kabutes.
  2. " - (dvigubos) Taip pat kaip ir su viengubomis kabutėmis.
  3. ` - (paverstos) Kai norime tekste panaudoti kintamuosius.

JS kintamieji (vanilla js)

  1. var - (variable) kintamųjų pavadinimai yra case sensitive - (var x = 'Pirmasis kintamasis';)
  2. string - paprastas tekstas - ( alert('Tai yra tekstas - string');)
  3. pakeičiame reikšmę kintamajam - (x = 'Antrasis kintamasis';)
  4. Sveikasis kaičius(integer) - (var y = 15;) ir string - (var y = '15';)
  5. Pirmoji funkcija -
    var x = 10;
    var result = x + 12;
    alert('10 + 12 = ' + result);
    console.log(result);

Užduotys

  1. Sukurkite tris kintamuosius, kurie saugotų tris reikšmes: Vardas, Pavardė, Gimimo metai ir naudojantis jais konsolėje atvaizduokite eilutę: “Aš esu Vardenis Pavardenis gimęs xxx”.
  2. Įsidėkite skripte šią funkciją funkciją:
    function randomSkaicius(min, max) {
    return Math.floor(Math.random() * (max - min + 1) + min)
    }
    Tai atsitiktinių skaičių generatorius. Jos panaudojimas būtų toks: randomSkaicius(0, 10) 0 ir 10 reikšmės nusako nuo kokio iki kokio skaičiaus turėtų būti norimas rezultatas.
  3. Sukurkite du kintamuosius ir naudodamiesi antrame punkte pateikta funkcija priskirkite jiems atsitiktines reikšmes nuo 0 iki 4. Su šiais dviem skaičiais atlikite keturias matematines operacijas: Sudėtis, atimtis, daugyba, dalyba.
  4. Pasinaudodami trečiame punkte pateikta funkcija, sukurkite tris kintamuosius ir priskirkite jiems tris reikšmes nuo 0 iki 25. Pasitelke Math.min() ir Math.max() metodus suraskite vidurinį skaičių.
